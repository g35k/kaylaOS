# KaylaOS

A personal portfolio website inspired by the macOS Aqua aesthetic (Leopard/Lion era, 2010–2012), built with React + Vite.

## Tech Stack

- React (Vite)
- React Router
- Font Awesome
- Google Fonts (Instrument Serif)

## Structure
```
src/
├── assets/
├── components/
│   ├── navbar/
│   │   ├── navbar.jsx
│   │   └── navbar.css
│   └── sidebar/
│       ├── sidebar.jsx
│       └── sidebar.css
├── pages/
│   ├── welcome.jsx
│   └── welcome.css
├── App.jsx
├── App.css
└── main.jsx
```

## Running Locally
```bash
npm install
npm run dev
```
